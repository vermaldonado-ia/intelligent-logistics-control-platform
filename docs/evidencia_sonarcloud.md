# 🧪 Evidencia de Calidad — SonarCloud

## 🎯 Propósito

Mostrar de forma transparente la calidad del código del proyecto mediante análisis automático, evidenciando cumplimiento de estándares básicos de desarrollo.

---

## 🧠 Enfoque

Se utiliza **SonarCloud** como herramienta de análisis continuo para evaluar:

* Calidad del código
* Cobertura de pruebas
* Detección de problemas potenciales

El objetivo no es alcanzar métricas avanzadas, sino asegurar un nivel mínimo de calidad y buenas prácticas.

---

## 🔍 Análisis realizado

El análisis incluye:

* ✔ Detección de bugs potenciales
* ✔ Identificación de code smells
* ✔ Cobertura de pruebas
* ✔ Validación de mantenibilidad

---

## 📊 Resultados generales

El proyecto ha sido evaluado mediante SonarCloud, obteniendo un estado de calidad que permite:

* Validar el código base
* Detectar mejoras potenciales
* Asegurar consistencia en el desarrollo
